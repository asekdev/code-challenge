# :computer: Coding Challenge

## Requirements

Write some code to consume the json and output a list of all the cats in alphabetical order under a heading of the gender of their owner.

#### Example Output

**Male**

- Angel
- Molly
- Tigger

**Female**

- Gizmo
- Jasper

## Installation

Install required dependencies;

```javascript
npm install
```

## Tests

Run tests;

```javascript
npm test
```

## Usage

Run program;

```javascript
npm start
```
